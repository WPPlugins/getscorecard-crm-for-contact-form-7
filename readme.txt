== Description ==

### GetScorecard CRM for Contact Form 7  | GetScorecard.com ###

This plugin extends Contact From 7 into a full CRM with a GetScorecard cloud based CRM. Supported and maintained by GetScorecard.com

[youtube https://www.youtube.com/watch?v=SUlz67hQMdU&feature=youtu.be]

### Getting Started  ###
1) Install Contact Form 7 plugin

2) Create a free account at GetScorecard.com

3) Install GetScorecard CRM for Contact Form 7  plugin and link to GetScorecard Account

4) Open Contact Form 7 - Select form to map - Map form fields via the GetScorecard Tab

Support is provided via www.support.getscorecard.com


### About ###

GetScorecard CRM for Contact Form 7

This plugin extends Contact Form 7 by linking it to the GetScorecard CRM Cloud software allowing you to fully manage contacts created via Contact Form 7.

GetScorecard is a CRM for small business that offers a free and paid CRM cloud based service. It includes Contact Management, Sales Management and Pipeline Management
